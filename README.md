## 📝 About This Repository

This repository contains solutions to competitive programming problems from multiple sources including **CSES Problem Set**, **USACO**, and various Vietnamese online judges. Each solution is organized by algorithm category and problem type, making it easy to find implementations for specific topics.

The collection includes:
- 📚 **200+ Problem Solutions** across various categories
- 🎯 **Algorithm Implementations** with explanations
- 🔄 **Multiple Approaches** for complex problems
- 🇻🇳 **Vietnamese & English Comments** in code

## 🎯 Key Algorithm Categories

### Core Algorithms
- ✅ **Sorting & Searching**: Quick sort, merge sort, binary search
- ✅ **Dynamic Programming**: Knapsack, LCS, LIS, coin change
- ✅ **Graph Algorithms**: DFS, BFS, Dijkstra, Floyd-Warshall
- ✅ **Number Theory**: Primes, GCD, Euler's theorem, modular arithmetic
- ✅ **Greedy Algorithms**: Activity selection, coin change, scheduling

### Advanced Techniques
- 🔄 **Bitmask DP**: Optimization using bit manipulation
- 🔄 **Binary Search**: Advanced searching and range queries
- 📋 **Data Structures**: Segment trees, Fenwick trees, DSU
- 📋 **String Algorithms**: KMP, hashing, pattern matching
- 📋 **Geometry**: Convex hull, line intersection, area calculation

## 💡 How to Use This Repository

1. **Browse by Category**: Navigate to `CP/` folder and select your algorithm category
2. **Search by Topic**: Most files include the algorithm name in the filename
3. **Review Solutions**: Each file contains well-commented C++ code
4. **Learn Patterns**: Compare different approaches to the same problem type
5. **Practice**: Use these as reference implementations for your own solutions

### Compilation & Execution
```bash
# Using g++ (C++17 standard)
g++ -std=c++17 -O2 -Wall solution.cpp -o solution
./solution < input.txt

# Or run directly with the Build and Run task in VS Code
```
